END-TO-END URDU CONVERSATION
Voice Input → AI Processing → Voice Output

System Flow:

1. User speaks in Urdu → 2. Frontend captures audio → 3. Backend processes through STT → 4. Voice biomarkers extracted → 5. NLP analyzes text → 6. RL agent selects optimal response → 7. CBT engine generates therapy response → 8. TTS converts to Urdu speech → 9. User receives voice output → 10. Feedback updates RL model
